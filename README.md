# DevOps-Server
DevOps Manager — это расширенный инструмент для управления портами Windows Firewall, локальным хостингом, SSH и автоматизации сетевых операций. Программа предоставляет удобный GUI (графический интерфейс) и REST API для управления портами и протоколами удалённого доступа. От Hack2p Team.
